# Titanic Dataset Analysis with Pandas

## 📌 Overview

This project analyzes the Titanic dataset using **Python** and **Pandas** to practice data cleaning, feature engineering, and exploratory data analysis (EDA). The analysis focuses on understanding passenger characteristics and the factors that influenced survival.

## 🛠 Technologies

* Python
* Pandas
* NumPy
* Jupyter Notebook

## 📊 Project Tasks

* Explored and summarized the dataset.
* Handled missing values in `Age` and `Cabin`.
* Created new features such as **AgeGroup** and **Title**.
* Calculated descriptive statistics and correlations.
* Sorted and filtered passenger data.
* Analyzed survival rates by **Sex**, **Pclass**, and **Title** using `groupby()` and aggregation functions.

## 📈 Key Findings

* Female passengers had a much higher survival rate than males.
* First-class passengers were more likely to survive than second- and third-class passengers.
* Passenger titles (e.g., **Mr**, **Mrs**, **Miss**, **Master**) provided additional insight into survival patterns beyond gender alone.
